# 🌸 Our Secret Space

A real-time, magical scrapbook Progressive Web App (PWA) built for me and my bestie. 

✨ **Features:**
* Real-time "Thinking of You" pings
* Synchronized mood tracking
* Live memory lane notes with timestamps
* Dual music players
